# urlshorterner
Self-hosted Nodejs url shorterner service

# installation
```
git clone https://github.com/jhfoo/urlshorterner.git
cd urlshortener
npm i
```

# config
Edit pass.txt with edit authorization password

# run modes
Developer mode
```
npm run dev
```

Production mode
```
npm start
```
